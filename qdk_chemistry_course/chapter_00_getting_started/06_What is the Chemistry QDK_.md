<h3>What is the Chemistry QDK?</h3>

The Chemistry QDK is a domain-specific library that sits on top of the core QDK tools. It adds chemistry-aware typed objects — `Structure`, `Wavefunction`, `Orbitals`, `Hamiltonian`, `QubitHamiltonian` — and a unified registry API for creating and configuring algorithms, so that each pipeline step produces and consumes strongly-typed chemistry objects rather than raw arrays. The full architecture is described in <a href="https://arxiv.org/abs/2601.15253" target="_blank">Baker et al. (2025)</a>.
