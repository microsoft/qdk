<h1 style="color:#D30982;text-align:center;">Advanced Chemistry Workflows on Magne </h1>

<p style="text-align: justify;">

Welcome to qBraid's course on the Microsoft Chemistry QDK. This course will equip you with the necessary skills and tools to use the Microsoft Chemistry QDK confidently. You will learn the complete workflow for <a href="https://arxiv.org/abs/1808.10402" target="_blank">quantum chemistry simulations</a>: starting with classical preprocessing techniques to reduce problem size (active space selection, Hamiltonian construction), then encoding molecular systems as qubits using efficient mappings, preparing quantum states, and implementing Quantum Phase Estimation (QPE) algorithms to calculate ground-state energies. The course emphasizes hands-on experience with QDK tooling through practical notebooks, culminating in a complete demonstration where you implement QPE on a model Hamiltonian. Rather than focusing on theoretical foundations, the curriculum prioritizes operational knowledge; you will learn how to actually use these tools for real computational chemistry workflows, with all techniques demonstrated using Microsoft's quantum software stack. The course is aimed at computational chemists, quantum computing researchers, and advanced students who want hands-on experience running a complete quantum chemistry workflow. A working knowledge of Python and basic quantum mechanics is assumed; no prior experience with quantum computing software is required. The course spans nine chapters and is designed to be completed in approximately 5-6 hours.
</p>
