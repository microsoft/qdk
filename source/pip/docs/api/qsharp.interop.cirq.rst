qsharp.interop.cirq
===================

The ``qsharp.interop.cirq`` module provides a Cirq ``Sampler`` backed by
the local Q# NeutralAtomDevice simulator, allowing Cirq circuits to be run
without a cloud connection.

.. note::
   Requires the optional ``cirq`` dependency. Install with:
   ``pip install qsharp[cirq]``
