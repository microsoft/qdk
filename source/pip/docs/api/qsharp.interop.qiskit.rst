qsharp.interop.qiskit
=====================

The ``qsharp.interop.qiskit`` module provides Qiskit backends backed by
the local Q# simulator, allowing Qiskit circuits to be run and estimated
without a cloud connection.

.. note::
   Requires the optional ``qiskit`` dependency. Install with:
   ``pip install qsharp[qiskit]``
