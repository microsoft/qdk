# qsci - Q# interactive command-line

```console
Q# Interactive

Usage: qsci [OPTIONS]

Options:
      --use <USE>      Use the given file on startup as initial session input
      --open <OPEN>    Open the given namespace(s) on startup before executing the entry expression or starting the REPL
      --entry <ENTRY>  Execute the given Q# expression on startup
      --nostdlib       Disable automatic inclusion of the standard library
      --exec           Exit after loading the files or running the given file(s)/entry on the command line
  -h, --help           Print help
  -V, --version        Print version
```
