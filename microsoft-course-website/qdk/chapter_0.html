The Microsoft Quantum Development Kit

The Quantum Development Kit is Microsoft's open-source toolkit for writing, simulating, and running quantum programs. It includes Q#, a language purpose-built for quantum computing; a suite of simulators from full state-vector to Clifford and sparse; the Azure Quantum Resource Estimator for projecting fault-tolerant hardware costs; and a Python and Jupyter interface that ties everything together. The same program can run on a local simulator today and compile to hardware-ready QIR tomorrow. This course covers the full QDK stack — from your first program through simulation backends, visualization, error correction, algorithms, chemistry applications, and compilation. Chapter 1 shows how to use GitHub Copilot as a quantum coding assistant from the start.

Chapter 0: Getting Started with the QDK

This chapter gets your environment ready and gives you the conceptual foundation for the rest of the course. By the end, you will have the QDK running locally and a clear picture of how the two execution modes fit together. The final cell in this chapter must pass before proceeding to Chapter 1.

What you'll learn

  • Set up VS Code with the QDK extension and verify your environment
  • Understand the two ways to run Q# in a notebook: the %%qsharp magic and the Python API
  • Run your first Q# program and observe quantum randomness firsthand
  • See how qsharp.run() collects multiple shots and what that reveals about entanglement
  • Preview the course structure and where each tool fits

How to access this course

This course is available in two modes — you will use both!

qBook (web browser)

qBook is the starting point for your learning journey. First, follow instructions in Chapter 0 to get your local set up ready. Answer the multiple-choice and free-response questions directly on the page. You will use qBook for all the learning material!

VS Code (local)

After following instructions in this chapter, you will have your local set up ready. You will run code cells and modify code locally. The Setup section below walks you through cloning the repository, installing the Azure Quantum Development Kit extension, and selecting your Python environment as the kernel.

You do not need both at once: qBook is self-contained for reading and answering questions; VS Code is self-contained for running code. If you are following along in qBook and want to run a cell interactively, switch to the corresponding notebook in VS Code.

Setup: local environment

Complete these steps before running any code. This is the only chapter that requires local setup — all other chapters run inside your configured environment.

Step 1: Install the VS Code QDK extension

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X) and search for "Azure Quantum Development Kit"
  3. Install it — this adds Q# syntax highlighting, IntelliSense, and inline circuit diagrams

Step 2: Clone the course repository

git clone https://github.com/qBraid/microsoft-course.git
cd microsoft-course

Step 3: Open the QDK course folder in VS Code

  1. Open the cloned microsoft-course folder in VS Code
  2. Navigate to qdk/course/chapter_00_getting_started.ipynb
  3. Select your Python environment as the kernel
  4. Run the verification cell below — it should print the installed version without errors

If you are also taking the Chemistry course, its notebooks are in chemistry/course/ — both courses live in the same repository.
