Chapter 4: Visualization

Visualization is the diagnostic layer that turns raw Q# simulation output into insight. This chapter covers the three main visualization surfaces in the QDK: circuit diagrams via qsharp.circuit(), state inspection via DumpMachine() and DumpRegister(), and measurement distribution plots via qsharp_widgets.Histogram and matplotlib. You will see how target profile affects circuit diagrams, when static synthesis fails and simulation-based generation is needed, and how circuits themselves serve as a first-pass debugging tool.

What you'll learn

  • Generate and render circuit diagrams with qsharp.circuit() and the Circuit widget
  • Use generation_method=CircuitGenerationMethod.Simulate for circuits with mid-circuit measurements
  • Understand how target profile affects circuit diagrams: Unrestricted vs Base
  • Visualize quantum state interactively with DumpMachine() and DumpRegister()
  • Plot measurement histograms with Histogram and matplotlib
  • Use circuit diagrams as a debugging tool to catch gate ordering mistakes
  • Distinguish static synthesis from simulation-based circuit generation

4.1 Circuit Diagrams: Basic Usage

qsharp.circuit(expr) synthesizes a circuit diagram for any Q# entry expression. Wrap it in Circuit() to get an interactive SVG rendering in Jupyter.

Pass a concrete call expression rather than an operation name when your operation takes Qubit[] arguments.
