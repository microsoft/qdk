Advanced Chemistry Workflows on Magne

Welcome to qBraid's course on the Microsoft Chemistry QDK. This course will equip you with the necessary skills and tools to use the Microsoft Chemistry QDK confidently. You will learn the complete workflow for quantum chemistry simulations: starting with classical preprocessing techniques to reduce problem size (active space selection, Hamiltonian construction), then encoding molecular systems as qubits using efficient mappings, preparing quantum states, and implementing Quantum Phase Estimation (QPE) algorithms to calculate ground-state energies. The course emphasizes hands-on experience with QDK tooling through practical notebooks, culminating in a complete demonstration where you implement QPE on a model Hamiltonian. Rather than focusing on theoretical foundations, the curriculum prioritizes operational knowledge; you will learn how to actually use these tools for real computational chemistry workflows, with all techniques demonstrated using Microsoft's quantum software stack.

Chapter 0: Getting Started

This chapter gets your environment ready and gives you the conceptual foundation for the rest of the course. By the end, you will have the Chemistry QDK running locally and a clear picture of how the full simulation workflow fits together. The final cell in this chapter must pass before proceeding to Chapter 1.

What you'll learn

  • How to set up your local environment: clone the course repo and install the Chemistry QDK (prereq gate)
  • Where the Chemistry QDK sits within the Microsoft quantum software stack
  • How the Chemistry QDK differs from the base QDK: domain-specific typed objects and a chemistry-first workflow
  • The algorithm registry: one interface to create, configure, and discover every component
  • What a complete end-to-end molecular simulation looks like, from structure input to energy output

How to access this course

This course is available in two modes вЂ” you will use both!

qBook (web browser)

You do not need both at once: qBook is self-contained for reading and answering questions; VS Code is self-contained for running code. If you are following along in qBook and want to run a cell interactively, switch to the corresponding notebook in VS Code.

Setup: local environment

Complete these steps before running any code in this course. This is the only chapter that requires local setup вЂ” all other chapters run inside your configured environment.

Step 1: Install VS Code

  1. Download and install Visual Studio Code
  2. Open VS Code and install the Python extension if not already present

Step 2: Install qdk-chemistry

In a terminal (inside your virtual environment): 
pip install 'qdk-chemistry[plugins, qiskit-extras, jupyter]'

Full installation instructions including system dependencies (LAPACK, HDF5): INSTALL.md

Step 3: Clone the course repository

In a terminal: 
git clone https://github.com/qBraid/microsoft-course.git
cd microsoft-course

This gives you all the course notebooks (with exercises) and the example data files used throughout.

Step 4: Open the course in VS Code

  1. In VS Code, open the cloned microsoft-course folder
  2. Open chemistry/course/chapter_00_getting_started.ipynb
  3. Select your Python environment as the kernel
  4. Run the verification cell below вЂ” it should print the installed version without errors

If you are also taking the QDK course, its notebooks are in qdk/course/ вЂ” both courses live in the same repository.
