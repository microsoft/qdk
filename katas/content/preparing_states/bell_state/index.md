**Input:** Two qubits in the $|00\rangle$ state (stored in an array of length 2).

**Goal:**  Change the state of the qubits to $|\Phi^{+}\rangle = \frac{1}{\sqrt{2}} \big (|00\rangle + |11\rangle\big)$.
